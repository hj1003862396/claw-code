# Rust 版启动说明

在 **`rust/`** 目录下执行（工作区根目录不是 Cargo 工作区）。

## 交互式 REPL（默认）

```bash
cd rust
cargo run -p rusty-claude-cli
```

不加参数会启动 `claw` 交互界面；编译产物位于 `target/debug/claw`，也可在构建后直接运行该二进制。

## 常用变体

```bash
# 查看帮助
cargo run -p rusty-claude-cli -- --help

# 指定模型
cargo run -p rusty-claude-cli -- --model claude-opus-4-6

# 单次提问后退出
cargo run -p rusty-claude-cli -- "你的问题"
```

## 校验（可选）

与仓库约定一致时，可在 `rust/` 下执行：

```bash
cargo fmt
cargo clippy --workspace --all-targets -- -D warnings
cargo test --workspace
```
